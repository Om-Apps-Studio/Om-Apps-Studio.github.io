Camera & Media Capture

Advanced Time Management

Time Synchronization

  • NTP Time Sync: Network Time Protocol synchronization for maximum accuracy
  • GPS Time Sync: GPS-based time when satellite signals are available
  • Device Time Fallback: Uses device system time when network unavailable
  • Time Source Priority: Automatic fallback chain: NTP ‚Üí GPS ‚Üí Device
  • Time Source Tracking: Records which time source was used for each capture
  • Automatic Re-sync: Periodic time synchronization every 5 minutes when conditions are met
  • Time Verification: Validates time consistency and restarts sync if needed

Time Display

  • UTC Timestamps: All timestamps stored in UTC for consistency
  • Localized Display: Formatted display time in local timezone
  • Time Source Indicator: Shows whether time came from NTP, GPS, or device
  • Format: DD/MM/YYYY HH:MM:SS format for easy reading
